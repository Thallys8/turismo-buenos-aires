# Anexo - "Frameworks y NodeJS"

Las Frameworks proveen a desarrolladores de herramientas y convenciones, probadas y testeadas, que faciliatan la creacion de aplicaciones web escalables e interactivas. Actuando como una base que mejora la experiencia de desarrollo al cubrir tareas comunes de este proceso.  
Dentro de las herramientas que traen estas frameworks, esta la principal que es la administracion de la interaccion con el DOM y el manejo de estados, a niveles que varian segun el framework. Pero tambien traen herramientas como el Routing, Builds, Renderizado de HTML e incluso formas de debuggear/testear el codigo del framework.  
Los dos stacks mas comunes de frameworks es el stack MERN (**M**ongoDB, **E**xpress, **R**eact y **N**ode), que es mas liviano al utilizar react como libreria front-end, y MEAN (**M**ongoDB, **E**xpress, **A**ngular y **N**ode), que es mas estructurado al ya contener herramientas de routing, validacion, etc. dentro del framework frontend Angular.
